- When the output says "<action>...", that's when the action starts. If it says it's action something after that line, you can assume that the action of what the prior line says is done. For example, if it's "Generating hpp code for src/translator.lua at 0.049279...", then "Writing transpiled output for src/translator.lua at 0.108216...", that means that the hpp code started generating at 0.049279 seconds and ended generating at 0.108216 seconds. The exception is if it explicitly says that an action is done.
